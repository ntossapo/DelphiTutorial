DelphiTutorial
==============

เขียนบน Delphi Xe5 เพราะฉนั้นเปิดกับ Embacadero Delphi XE นะครับ
เปิดกับไฟล์นามสกุล groupproj เปิดมาแล้วจะมีโปรเจ็คแต่ละเรื่องขึ้นมาเลย
แล้วเลือก compile เอาว่าจะ ดูตัวไหน
